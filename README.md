# Functional Programming Exam Template
Simple LaTeX template for the Functional Programming (02157) Winter Exam 2025.

## Setup
1. Edit `main.tex` and add your name and student ID:
   ```latex
   \newcommand{\myname}{YOUR NAME HERE}
   \newcommand{\mystudentid}{YOUR STUDNET ID HERE}
   ```

2. Write your F# code in the `Code/` folder:
   - `Question1.fsx`
   - `Question2.fsx`
   - `Question3.fsx`
   - `Question4.fsx`
   - `Question5.fsx`

## Compile
```bash
pdflatex main.tex
```
